
# ZapShift

A modern courier service web application built with React, Node.js, and MongoDB.

## Features

- **Authentication**: Google Sign-in, Email/Password login via Firebase
- **Password Recovery**: Forgot password and OTP verification system
- **Role-Based Dashboards**: Separate interfaces for Admin, User, and Rider
- **Send-Parcel** : Send Parcel document/non document to same/different district 
- **Payment Gateway**: Stripe integration for secure transactions
- **Parcel Tracking**: Real-time tracking of courier shipments
- **Rider Management**: District-based rider assignment and optimization
- **User Management**: Complete user profile and account management
